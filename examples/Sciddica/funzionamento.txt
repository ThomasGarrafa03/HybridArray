COME FUNZIONA L'AUTOMA CELLULARE SCIDDICA?

01. INFORMAZIONI
Ogni cella (i,j) dell'automa presenta tre informazioni:
        - z: la quota del terreno
       	- h: l'altezza del fluido (spessore)
       	- f(dir): il flusso verso la direzione dir

02. STEP DI TRANSIZIONE
Ad ogni step di transizione, ogni singola cella determinerà
	1. quanta parte di fluido si sposta verso ciascun vicino
        2. aggiorna le altezze h 
secondo questa formula:

nuovo_h(i,j) = h(i,j) + somma(flussi entranti) - somma(flussi uscenti)

03. CALCOLO FLUSSI

Considero tutti i vicini della cella (4 per come è stato definito in SciddicaImpl.h) e l'elemento centrale. Ne computo quindi l'altezza totale:
double w[5]; // 4 vicini + cella centrale
w[0] = z[i][j] + h[i][j];         // cella corrente
w[1] = z[i-1][j] + h[i-1][j];     // nord
w[2] = z[i+1][j] + h[i+1][j];     // sud
w[3] = z[i][j+1] + h[i][j+1];     // est
w[4] = z[i][j-1] + h[i][j-1];     // ovest
